
# Zara- Business- Sales Analysis (Interactive  Dashboard created using MS Excel)
## Project Objective
To transform raw retail sales data into a structured, insight-driven dashboard using Excel (Pivot Tables, KPI metrics, slicers, and visualizations) to support strategic business planning.

## Dataset used
-	<a href=“https://github.com/Shivanidere13/Data-Analysis-Dashboard/blob/main/zara_business_sale%20dashboard.xlsx”>Dataset</a>

## Question(KPIs)
•	What is the total revenue generated?
•	What is the total number of units sold?
•	Which season shows peak sales performance?
•	Which category contributes the most to total volume?
•	Which are the Top 10 best-selling products?
•	Are promotions impacting profitability?
•	Which material generates the highest revenue?
•	Do customers prefer comfort materials (Cotton/Wool) over luxury fabrics (Silk/Satin)?
•	Are promotions impacting profitability?
•	Which product position (Aisle, End-cap, Front) generates maximum sales?
